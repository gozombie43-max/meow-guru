# 🧠 Meow Guru – Smart Quiz Platform

Meow Guru is a modern web-based quiz application designed for structured exam preparation, especially for competitive exams like SSC CGL. It supports math-heavy questions, diagrams, and clean UI for efficient learning.

---

## 🚀 Features

- 📚 Structured Quiz System (MCQs)
- 🧮 KaTeX-based Math Rendering
- 🖼️ Image + Question Integration (for geometry & diagram-based questions)
- ⚡ Fast and Responsive UI
- 📊 JSON / NDJSON-based Question Database
- 🎯 Focused on Competitive Exam Preparation

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React
- **Styling:** Tailwind CSS / Custom CSS
- **Math Rendering:** KaTeX
- **Data Handling:** JSON / NDJSON
- **Version Control:** GitHub
